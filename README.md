# javascriptSlimeSoccer

Live link: https://arig501.github.io/javascriptSlimeSoccer/

A simple, fast paced two-player slime soccer game made using JavaScript. 

**Player 1 controls:** 

A - move left

D - move right

W - jump

S - Catch

**Player 2 controls:**

Left Arrow - move left

Right Arrow - move right

Up Arrow - jump

Down Arrow - Catch
